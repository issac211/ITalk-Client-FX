# ITalk-Client-FX (Client)

## Overview
ITalk-Client-FX is the **client-side** application for the ITalk forum system, developed as part of the **"Programming in the Internet Environment"** course. This project provides a graphical user interface (GUI) for users to interact with the ITalk server, send and receive messages, and search through forum discussions.

For the server-side project visit [ITalk-Project](https://github.com/issac211/ITalk-Project/tree/main)

## Features
- **Interactive GUI** – built using JavaFX.
- **Seamless server communication** – connects to the ITalk server via TCP/IP.
- **Real-time messaging** – supports sending and receiving messages.
- **Text search functionality** – allows users to search for messages in the forum.
- **Multi-threaded client handling** – ensures smooth user experience.

## Technologies Used
- **Java** – primary programming language.
- **JavaFX** – for building the graphical user interface.
- **TCP/IP** – client-server communication protocol.
- **OOP and Design Patterns** – follows the **MVC (Model-View-Controller)** architecture.
